# Kingdom Home - Project Specification

## Overview

Create a landing page for the omyclaude portfolio that serves as a hub linking to all assignment projects, built as a single HTML file with embedded CSS.

## Core Requirements

### 1. Visual Design
- Dark gradient background with a modern, clean aesthetic
- Responsive layout that works on all screen sizes
- Smooth hover animations for interactive elements

### 2. Header Section
- Display gamertag (omyclaude) as main title
- Subtitle showing kingdom/organization name
- "Hello, World!" greeting message

### 3. Project Cards
- Display all portfolio projects as clickable cards
- Each card shows project name and brief description
- Cards link to their respective project folders
- Hover effects for visual feedback

### 4. Footer
- Simple footer with course identifier

## Technical Specifications

### Technologies
- HTML5 semantic structure
- CSS3 for styling (gradients, flexbox, transitions)
- No JavaScript required
- No external dependencies

### Visual Style
- Color palette: Deep blues (#1a1a2e, #16213e, #0f3460)
- Accent color: Coral red (#e94560)
- Text: White primary, gray secondary (#a0a0a0)
- Cards: Semi-transparent white backgrounds

### Responsive Design
- Flexbox layout with wrapping
- Centered content with max-width constraints
- Mobile-friendly padding and font sizes

## Deliverables

1. Single `index.html` file at repository root
2. Deployed to GitHub Pages as kingdom home
3. spec.md with project specification
4. README.md with project documentation
