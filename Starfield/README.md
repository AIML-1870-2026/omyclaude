# Starfield Cockpit - Flight Deck

A single-file HTML/CSS/JS interactive starfield simulation viewed from a spaceship cockpit interior.

## Live Demo

**[View Live Demo](https://abrahmanzai.github.io/Assignment-2/)**

## Features

- **3D Particle Starfield**: Warp-speed star simulation with trail effects
- **Cockpit Interior**: Multi-layered CSS architecture with canopy pillars, ship frame, and dual flight yokes
- **Industrial UI**: "Used-future" aesthetic with glowing HUDs and CRT scan-line effects
- **Interactive Controls**:
  - **Velocity**: Adjust thrust/travel speed
  - **Warp Factor**: Control field of view depth
  - **Density**: Change star particle count
  - **System Reset**: Restore default settings

## Technologies Used

- HTML5 Canvas for particle rendering
- CSS3 with custom properties, gradients, and animations
- Vanilla JavaScript (no frameworks)
- Tailwind CSS for utility styling
- Google Fonts (Orbitron, JetBrains Mono)

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/abrahmanzai/Assignment-2.git
   ```
2. Open `index.html` in a web browser
