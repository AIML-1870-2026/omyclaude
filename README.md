# omyclaude - Kingdom Home

A portfolio landing page for AIML 1870, serving as a hub for all assignment projects.

## Live Demo

**[View Live Demo](https://aiml-1870-2026.github.io/omyclaude/)**

## Features

- **Hello World Landing Page**: Clean, modern entry point for the portfolio
- **Project Cards**: Quick access to all assignments with hover effects
- **Responsive Design**: Works on desktop and mobile devices
- **Dark Theme**: Elegant gradient background with accent colors
